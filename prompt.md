# PLN ULP Company Profile Website

## Overview

Design a modern **single-page company profile website** for an Indonesian electricity service office (**PLN Unit Layanan Pelanggan**).

The design should feel:

- Contemporary
- Trustworthy
- Clean
- Technology-driven
- Friendly and accessible for public users

---

## Color Palette

Use PLN-inspired colors:

| Role | Color |
|--------|--------|
| Primary Blue | #005BAC |
| Light Blue | #00AEEF |
| Accent Yellow | #FFD100 |
| Background | #FFFFFF |
| Section Background | Soft Gray (#F5F8FC) |

---

## Design References

Inspired by:

- Apple
- Stripe
- Linear
- Modern GovTech Websites

Avoid:

- Traditional image sliders/carousels
- Crowded layouts
- Large text-heavy sections
- Boxed and outdated government website designs

# Visual Direction

## Design Language

- Modern Enterprise UI
- Minimalist
- Premium Government Technology Platform

## UI Characteristics

- Large spacing
- Rounded corners (16–24px)
- Subtle shadows
- Glassmorphism navbar
- Bento-grid layouts
- Soft gradients
- Clean typography hierarchy

## Animations

- Smooth scrolling
- Framer Motion-style transitions
- Fade-in on scroll
- Card hover interactions
- Counter animations for statistics

## Responsiveness

- Mobile-first approach
- Fully responsive across all screen sizes
- Accessibility-focused
- Fast loading and optimized performance

---

## Final Impression

Create a website that feels like a combination of:

- Apple's cleanliness
- Stripe's modern professionalism
- Linear's simplicity
- Contemporary GovTech innovation

The final result should look like a premium public-service digital platform for 2026 rather than a traditional government website.
